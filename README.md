# GNU-MAKE READER

## Description
	
	Tool for reading makefiles and extracting files involved, useful if you want makefile functionality in your IDE.

## Dependencies:
	glib & makeheaders (http://www.hwaci.com/sw/mkhdr/)(https://aur.archlinux.org/packages/makeheaders/)

## Compile with:

```shell
make -j
```

## Test (& compile) with:

```shell
make run -j
```
